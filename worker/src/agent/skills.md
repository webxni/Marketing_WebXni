# Agent Skills

## What I can do

### Posts
- List, filter, search posts by client, date, status, platform
- Create a post targeting one specific platform
- Update post fields (title, caption, date, status, platforms)
- Bulk update posts matching filters
- Change post workflow status (approve, reject, ready, cancel)
- Publish a single post immediately
- Run bulk posting automation
- Fix failed posts by resetting to ready
- Delete a post
- Update blog post fields (content, SEO, keyword, slug)
- Publish/sync a blog post to WordPress
- Generate AI content for clients

### Clients
- Get full client details (profile, platforms, intelligence, services, areas)
- Update client profile fields (name, phone, industry, notes, brand, WordPress config)
- Update client intelligence (brand voice, tone, keywords, content goals)
- Update client platform config (upload-post ID, page ID, etc.)
- Add a new service to a client
- Add a service area to a client
- Add client feedback

### Offers & Events (GBP)
- Create a GBP offer for a client
- Update an existing offer
- Create a GBP event for a client
- Update an existing event

### Queue & Automation
- Show the posting queue (overdue, due soon, queued)
- Show generation run status
- Trigger a generation run
- Trigger bulk posting
- Fetch/sync published URLs

### System & Reports
- Get system health status (failed posts, stuck jobs, stale records)
- Get post statistics by client/date/status
- Suggest fixes for system issues
